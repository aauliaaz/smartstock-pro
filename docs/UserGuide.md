# Panduan Pengguna - SmartStock Pro

## 1. Login Ke Sistem
1. Buka browser dan masukkan URL aplikasi.
2. Masukkan Email dan Password Anda.
3. Klik **Sign In**.

## 2. Manajemen Produk
- **Melihat Daftar:** Klik menu **Products** di sidebar.
- **Menambah Produk:** Klik tombol **Add Product**, isi formulir, upload gambar, lalu simpan.
- **Pencarian:** Gunakan kotak pencarian di atas tabel produk untuk mencari berdasarkan Nama atau SKU.

## 3. Transaksi Inventaris
- **Barang Masuk:** Klik menu **Stock In**, pilih gudang, produk, dan masukkan jumlah.
- **Barang Keluar:** Klik menu **Stock Out**, pilih gudang, produk, dan masukkan jumlah (Sistem akan mengecek ketersediaan stok).

## 4. Transfer Antar Gudang
1. Klik menu **Transfers**.
2. Klik **Request Transfer**.
3. Pilih gudang asal dan tujuan, serta barang yang akan dipindahkan.
4. Manajer akan menerima notifikasi untuk menyetujui transfer tersebut.

## 5. Dashboard Monitoring
- Pantau total stok dan nilai inventaris di ringkasan statistik.
- Gunakan peta interaktif untuk melihat sebaran stok di seluruh Indonesia.
- Perhatikan kartu **System Status** untuk memastikan performa server optimal.
