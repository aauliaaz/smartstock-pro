# FAQ - Pertanyaan Umum

1. **Apa itu SmartStock Pro?**
   Sistem Manajemen Inventaris berbasis web untuk mengelola stok barang di berbagai gudang secara real-time.

2. **Bagaimana cara login jika lupa password?**
   Hubungi Administrator sistem untuk melakukan reset password melalui database.

3. **Apakah saya bisa memindahkan barang antar gudang?**
   Ya, gunakan fitur "Transfer Gudang" di menu Transaksi.

4. **Siapa yang bisa menyetujui transfer barang?**
   Hanya user dengan role Admin atau Manajer Gudang.

5. **Apa itu fitur "Stok Menipis"?**
   Fitur peringatan otomatis jika jumlah stok produk di bawah ambang batas (threshold) yang ditentukan.

6. **Bagaimana cara menambah kategori baru?**
   Masuk ke menu Inventaris > Kategori dan klik tombol "Tambah Kategori".

7. **Apakah sistem mendukung multi-gudang?**
   Ya, SmartStock Pro dirancang untuk mengelola stok di banyak lokasi gudang sekaligus.

8. **Dapatkah saya mengimport data produk dari Excel?**
   Ya, gunakan fitur Import di menu Data & Laporan.

9. **Apa itu Audit Log?**
   Catatan setiap aktivitas yang dilakukan user di sistem untuk keperluan keamanan dan pelacakan.

10. **Bagaimana cara melihat performa server?**
    Admin dapat melihatnya melalui menu Sistem > Monitoring Server.
